## 📌 Project Overview

**Lauki Finance: Credit Risk Modelling** is a machine learning application that evaluates the credit risk of loan applicants based on financial and behavioral attributes.

The system predicts whether a borrower is likely to default using key indicators such as income, loan amount, repayment behavior, and credit utilization.

The project demonstrates how machine learning can assist financial institutions in **risk assessment and lending decisions**.

---

## 🎯 Objective

Financial institutions must carefully assess the creditworthiness of applicants before approving loans.

This project aims to:

- Predict **credit risk probability**
- Identify **high-risk borrowers**
- Assist lenders in making **data-driven loan decisions**

## 📊 Input Parameters

The model evaluates risk using the following borrower details:

- Age
- Annual Income
- Loan Amount
- Loan Tenure
- Average Days Past Due (DPD)
- Delinquency Ratio
- Credit Utilization Ratio
- Number of Open Loan Accounts
- Residence Type
- Loan Purpose
- Loan Type

Derived Feature:
- **Loan to Income Ratio**

---

## 🧠 Machine Learning Logic

The model analyzes borrower characteristics and financial behavior to estimate credit risk.

Key factors influencing prediction:

- Borrower income stability
- Existing debt exposure
- Payment delays (DPD)
- Credit utilization
- Loan purpose and type

The system transforms inputs into numerical features and feeds them into a trained ML model for prediction.

## 🚀 Running Locally

Clone the repository

```
git clone https://github.com/KrishnaTiwari1cr/ml-project-credit-risk-model
```

Navigate to the project folder

```
cd ml-project-credit-risk-model
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run main.py
```
